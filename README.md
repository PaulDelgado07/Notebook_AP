# Apuntes de Aprendizaje Profundo y Reforzado

Este repositorio contiene notebooks desarrollados como parte de las clases de **Aprendizaje Profundo (Deep Learning)** y **Aprendizaje por Refuerzo (Reinforcement Learning)**.

## Contenido

- Desarrollo de modelos de redes neuronales
- Ejercicios prácticos
- Visualización de resultados
- Explicaciones paso a paso

## Objetivo

El propósito de este repositorio es servir como material de apoyo para el aprendizaje y la práctica de técnicas modernas de inteligencia artificial.

## Tecnologías utilizadas

- Python
- Jupyter Notebook
- Librerías de Machine Learning y Deep Learning

## Nota

Los notebooks pueden incluir gráficos, imágenes y ejemplos prácticos utilizados durante las clases.
